通过参考http://blog.jobbole.com/108668/自己写出的单点登录程序，首先要感谢这篇博客提供的思路。

一、项目说明：
1、由于是在本地测试，为了防止端口冲突，所以myserver项目部署在tomcat的端口号改成了8888，mysys1和mysys2项目部署在tomcat的端口号还是8080。
2、myserver属于服务端验证用户登录及退出，用到了redis数据库来缓存用户登录相关信息，默认是用户名“test”和密码“123”即可验证成功，后期如果想使用次程序需要自己再做修改连接到真正地数据库来查询用户名和密码。
3、mysys1和mysys2属于两个子系统，只要这两个系统任意一个登录，那么访问其他系统不需要用户再登录。退出也一样。
4、本项目用到了cookie，所以需要浏览器没有禁用cookie才可以顺利运行。用到了框架就是spring和springMVC，同时spring和redis集成。
5、最后，项目描述可能有很多不完善的地方，纯属自己闲来研究单点登录，希望勿喷，项目中有很多需要改进的地方，不过感觉可以当作参考，目前项目是可以运行的，基本的单点登录也实现了。
